# Animated Web Page Templates

A collection of single-page front-end templates built with HTML and CSS, focused on smooth animations and modern interface design. Developed as part of freelance web development work to demonstrate front-end styling, layout, and animation techniques.

## Live Demos

- **Template 1 — FALAK MUSCAT. This template is designed to suit the needs of industry leading companies looking to trade and invest accross international markets and businesses.**: `https://KrishParmar2804.github.io/web-templates/template-1/`
- **Template 2 — VCLEAN. This template is for a cleaning company used to take orders and contracts for high profile cleaning and management.**: `https://KrishParmar2804.github.io/web-templates/template-2/`

## Features

- Single-page responsive layouts
- Custom CSS animations and transitions
- Animated buttons for Login and URL headers
- Clean, semantic HTML structure

## Tech Stack

- HTML
- CSS

## Running Locally

1. Clone this repository:
   ```
   git clone https://github.com/KrishParmar2804/web-templates.git
   ```
2. Open `template-1/index.html` or `template-2/index.html` directly in your browser.
